## Packages
framer-motion | Essential for cinematic entrance animations and page transitions
date-fns | For date formatting and manipulation in the calendar view
react-day-picker | For the calendar date selector component
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind CSS classes safely

## Notes
The application requires a "Royal Futuristic" aesthetic with dark backgrounds and gold accents.
Fonts required: 'Cinzel' (headers) and 'Inter' (body).
API endpoints are defined in shared/routes.ts and use Zod for validation.
Routes needed: /, /market/:region (USA, India, Japan, Crypto).
